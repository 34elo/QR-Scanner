# QR Scanner TSD

Android-приложение для сканирования штрих-кодов с возможностью сохранения и выгрузки на Яндекс.Диск.

## Возможности

- Сканирование штрих-кодов через ATOL Scanner API
- Сохранение отсканированных кодов в CSV или XLSX
- Проверка кодов по загруженному файлу
- Выгрузка файлов на Яндекс.Диск
- Настройка префикса имени файла, типа файла, обрезки кода, дубликатов

## Настройка

1. Скопируйте `secrets.properties.example` в `secrets.properties`
2. Добавьте ваш токен Яндекс.Диска:

```properties
YANDEX_TOKEN=ваш_токен
```

## Сборка

```bash
./gradlew assembleDebug
```

APK будет в `app/build/outputs/apk/debug/`
